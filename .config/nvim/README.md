 - Telescope (+ dependencies)
 - Treesitter
 - nvim-cmp  
 - nvim-lspconfig
 - mason
 - gitsigns and lazygit
 - nerdtree
 - Vimtex
 - lualine (lualine config at lua/statusline.lua)
 - tabby (tabby config at lua/tabby.lua)
 - vimwiki + taskwiki
 - whichkey
 - others...
